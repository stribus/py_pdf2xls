# PDF to XLS Converter

Projeto desenvolvido para a extração de dados a partir de arquivos PDF, com a posterior conversão dos mesmos em planilhas Excel e armazenamento em banco de dados. 

Este projeto foi inspirado pelo vídeo do canal "Jornada de Dados", disponível em: [https://www.youtube.com/watch?v=vzpc4JowjbI](https://www.youtube.com/watch?v=vzpc4JowjbI)

## Sobre os Arquivos

- **pdf_viz.py**: Responsável por auxiliar na visualização do PDF, facilitando o desenvolvimento das marcações usadas pelo Camelot.
- **extractRedrex.py**: Utilizado para processar PDFs com padrão "Redrex", extraindo dados e convertendo-os para Excel.
